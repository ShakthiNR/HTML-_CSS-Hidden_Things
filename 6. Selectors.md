

## Targeted Selector

    ```html
    <div>
        <p>Hello, world!</p>
                <ul>
                <li> <p>1</p></li>
                <li>2</li>
                </ul>
        <p>Thank You</p>
    </div>
    ```


    ``` 
    1. div p{} - all p under div, 
        - (ie) Hello, world!, 1, Thank You

    2. div > p{} - only direct child of div 
        - (ie) Hello, world!

    3. div+p{} - directly after(next child) 
    ```

    ```html
    <form>
        <input type="text" placeholder="Username"/>
        <input type="password" placeholder="Password"/>
        <button>Submit</button>
    </form>
    ```

    ```
    Then,
    1. input{} - select all input fields
    2. input[type="text"], input[type="password"] - select particular input fields
    ```

## Pseudo-Selectors (:) - nth

    ```html
    <ul>
        <li>Item 1</li>
        <li>Item 2</li>
        <li>Item 3</li>
        <li>Item 4</li>
        <li>Item 5</li>
        <li>Item 6</li>
        <li>Item 7</li>
        <li>Item 8</li>
        <li>Item 9</li>
    </ul>
    ```



  ```css
  <style>
      li {
        padding: 0.25rem;
        margin: 0.25rem;
        list-style: none;
      }

      /* first-child */
      li:first-child {
        background: red !important;
      }

      /* last-child */
      li:last-child {
        background: blue;
      }

      /* nth-child selector matches every element that is the nth child, regardless of type, of its parent. */

      /* Position 3 */
      li:nth-child(3) {
        background: purple;
      }

      /* Every 3rd and 0 is the offset - From that Offset style every 3rd child  */
      li:nth-child(3n + 0) {
        background: orange;
      }

      /* Every 3rd after 7 */
      li:nth-child(3n + 7) {
        background: yellow;
      }

      /* odd */
      li:nth-child(odd) {
        background: #ccc;
      }

      /* even */
      li:nth-child(even) {
        background: #ddd;
      }
    </style>

```
